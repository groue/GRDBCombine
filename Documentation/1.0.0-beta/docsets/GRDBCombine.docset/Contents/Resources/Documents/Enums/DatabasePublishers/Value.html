  • scheduling(_:)

    Returns a publisher which starts the observation with the given ValueObservation scheduler.

    For example, scheduling(.immediate) notifies all values on the main queue, and the first one is immediately notified when the publisher is subscribed:

    let cancellable = observation
    .publisher(in: dbQueue)
    .scheduling(.immediate) // <-
    .sink(
        receiveCompletion: { completion in ... },
        receiveValue: { players: [Player] in
            print("fresh players: \(players)")
        })
// <- here "fresh players" is already printed.
